# ClashRoyaleAnalyticsWebApp

Full-stack Clash Royale deck analytics app with:
- Drag-and-drop deck builder (Evo / Wild / Hero-Champion slots)
- Tower troop selection + optimization
- Matchup simulation and deck scoring breakdown
- C# backend source and JavaScript frontend

Live app: https://clashroyaleanalyticswebapp.vercel.app

## Stack
- Frontend: HTML, CSS, JavaScript
- API (deployed): Vercel serverless functions
- Backend source: ASP.NET Core (C#)

## Security
- No API token is committed in this repository.
- Secrets should be stored in environment variables only.
